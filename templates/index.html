Student Math Performance Prediction Using Machine Learning

This project is a full-stack machine learning application that predicts a student's math score based on demographic and academic features. It demonstrates the application of regression models to real-world educational data and showcases end-to-end model deployment using Flask.

What It Does

  • Predicts a student’s math score out of 100 using supervised machine learning.
  • Inputs include gender, ethnicity, parental education, lunch type, test preparation, and language scores.
  • Outputs a predicted math score through a web-based form interface.

Technical Overview

  • Frontend: HTML/CSS with a responsive layout
  • Backend: Flask (Python)
  • Model: Trained regression model using scikit-learn
  • Dataset: Student performance dataset from Kaggle

Highlights

  • Implemented complete ML workflow: data cleaning, training, evaluation, deployment
  • Achieved ~85% R² score on test data
  • Designed with modularity and clarity for maintainability
